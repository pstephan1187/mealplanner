# Feature 6 - Print and Export

## Goal
Provide printable versions of the meal plan and shopping list.

## Scope
- Printable meal plan summary (single page).
- Printable shopping list checklist.

## UI/UX Notes
- Use clean, minimal layout with readable typography.
- Hide navigation chrome in print views.
- Ensure print routes work well on mobile and desktop.

## Deliverables
- Dedicated print routes and Inertia pages with print styles.
